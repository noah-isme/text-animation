## üí° Tips Memilih Animasi

### 1. **Performance First**
- CSS animations > JavaScript animations (untuk performa)
- Hindari terlalu banyak animasi sekaligus
- Gunakan `will-change` untuk optimasi

### 2. **Konteks adalah Kunci**
- Formal/professional ‚Üí minimalis (fade, slide)
- Creative/artistic ‚Üí unik (particle, morph)
- Tech/modern ‚Üí futuristic (hologram, glitch)
- Premium/luxury ‚Üí metallic, glass, glow

### 3. **Accessibility**
- Sediakan opsi `prefers-reduced-motion`
- Jangan gunakan animasi terlalu cepat/flashing
- Pastikan teks tetap readable

### 4. **Mobile Considerations**
- Animasi 3D berat ‚Üí gunakan versi simplified
- Touch interactions berbeda dari hover
- Test performa di device sesungguhnya

### 5. **Subtlety > Overload**
- Satu animasi kuat > banyak animasi biasa
- Reserve animasi dramatis untuk hero/key moments
- Animasi background harus subtle
