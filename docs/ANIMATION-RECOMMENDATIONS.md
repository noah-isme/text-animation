## üí° Animation Selection Tips

### 1. **Performance First**
- CSS animations > JavaScript animations (for performance)
- Avoid too many simultaneous animations
- Use `will-change` for optimization

### 2. **Context is Key**
- Formal/professional ‚Üí minimalist (fade, slide)
- Creative/artistic ‚Üí unique (particle, morph)
- Tech/modern ‚Üí futuristic (hologram, glitch)
- Premium/luxury ‚Üí metallic, glass, glow

### 3. **Accessibility**
- Provide `prefers-reduced-motion` option
- Don't use overly fast/flashing animations
- Ensure text remains readable

### 4. **Mobile Considerations**
- Heavy 3D animations ‚Üí use simplified versions
- Touch interactions differ from hover
- Test performance on actual devices

### 5. **Subtlety > Overload**
- One strong animation > many mediocre ones
- Reserve dramatic animations for hero/key moments
- Background animations should be subtle
